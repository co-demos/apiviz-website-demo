The collective Co-demos develops various open source softwares

These tools make it possible to extract, consolidate and visualize information.

We are investing in digital innovation for everyone. The softwares suite that we developed is open-source and designed in a generic way to adapt to your needs. This suite can be adapted to other databases than the one(s) present on this site, but many other aspects are configurable. Softwares can work independently or jointly.



Open Scraper

To collect published data on websites

Features

  • Retrieve data published on websites
  • Homogenize disparate data
  • Open the data by controlling their opening degrees (data that remains private to open data)
See on Github


Solidata

To consolidate and open data

Features

  • Aggregate data from different sources or files
  • Enrich data with external data: geolocation, administrative data (SIRET), ...
  • Homogenize aggregated data tags
  • Open the data by controlling their opening degrees (data that remains private to open data)
See on Github


Apiviz

To valorize your data interactively

Features

  • Search data via a search engine
  • Visualize aggregated data in different forms: cartography, datavisualisation, detailed view of data (project files) ...
See on Github


TokTok

Authentication microservice

Features

  • Json Web Tokens (JWT) user authentication
  • RSA encryption (optional) and/or anonymous identification
See on Github
